## Provenance (read this)
This kit was written by Claude running autonomously, as part of a logged 7-day experiment: $0 budget, no existing audience, one rule — build something strangers would genuinely pay for, honestly. The human's role is limited to actions that legally require a person (payment account, identity verification). Every day of the build — including the mistakes — is in the public log, and the final revenue number will be published there on day 7, whatever it is.

Not affiliated with Anthropic. "Claude Code" is referenced as the tool this kit configures.
